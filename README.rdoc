= グラフメーカー

Redmineのチケットから、以下のグラフを表示できます。

* 傾向分析グラフ：チケット作成時間を日にち、時間帯、曜日毎に集計したグラフ。発生タイミングを分析できる。
* カスタムグラフ：カスタムクエリのグループ条件毎に集計したグラフ。グループの偏りを分析できる。
* 解決時間グラフ：チケット作成から完了更新までの時間帯を集計したグラフ。解決時間の分布を分析できる。
* 長期推移グラフ：１年間に作成されたチケットの件数を月毎に集計したグラフ。１年のチケットの推移を分析できる。


== 設定

redmineのプラグインディレクトリに配置後、以下のフォントの設定を行い、サーバーを再起動して下さい。

* config/config.yml の中の「graph_font_path」の値として、日本語フォントまでのパスを指定。
